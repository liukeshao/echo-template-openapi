响应格式说明

所有API响应都使用统一的格式，HTTP状态码固定为200，通过 code 字段区分成功和失败：

  • code = 0: 请求成功
  • code != 0: 请求失败，具体错误码请参考错误码表

认证说明

需要认证的接口使用 Bearer Token 认证，在请求头中添加：

Authorization: Bearer <access_token>

获取用户列表

分页获取用户列表，支持按状态筛选和关键词搜索

Authorizations:
BearerAuth
query Parameters
page
integer >= 1
Default: 1
Example: page=1

页码，从1开始

page_size
integer [ 1 .. 100 ]
Default: 20
Example: page_size=20

每页数量，最大100

status
string
Enum: "active" "inactive" "suspended"
Example: status=active

按状态筛选用户

keyword
string
Example: keyword=john

搜索关键词，用于搜索用户名或邮箱

Responses

开发环境

http://localhost:8000/api/v1/users

https://api.example.com/api/v1/users
